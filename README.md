# Sistema de Cadastro de Estudantes em Java

## Descrição
Este projeto é um exemplo simples desenvolvido em Java com o objetivo de demonstrar conceitos básicos de Programação Orientada a Objetos (POO), como:

- Criação de classes
- Uso de construtores
- Encapsulamento com métodos getters e setters
- Entrada de dados com Scanner
- Organização de código em múltiplas classes

O sistema permite criar um objeto do tipo `Estudante` e preencher seus dados via entrada do usuário.

---

## Estrutura do Projeto

- `Main.java`  
  Classe principal responsável por iniciar o programa e executar as ações.

- `Estudante.java`  
  Classe que representa um estudante, contendo atributos, construtores, getters, setters e métodos auxiliares.

---

## Funcionalidades

- Cadastro de estudante via teclado
- Armazenamento de:
  - Nome
  - Idade
  - Matrícula
  - Endereço
- Métodos simulando ações do estudante:
  - Estudar
  - Realizar atividades
  - Fazer provas

---

## Tecnologias Utilizadas

- Java
- Scanner (entrada de dados)

---

## Objetivo Educacional

Este projeto foi desenvolvido para fins didáticos, sendo ideal para iniciantes que estão aprendendo:

- Sintaxe básica do Java
- Diferença entre classes e objetos
- Uso de métodos
- Boas práticas como encapsulamento
